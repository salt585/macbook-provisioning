## mas
Mac App StoreにあるアプリをCUIでインストールすることができるツール。

### Sign-in
```
mas signin APPLE_ID
Password:
```

### commands
```
# インストールしたパッケージを確認
mas list

# パッケージを検索
mas search XXX

# インストール
mas install [APP_ID or APP_NAME]

# パッケージの更新の確認
mas outdated

# 全パッケージの更新
mas upgrade

# 特定のパッケージの更新
mas upgrade APP_ID
```
